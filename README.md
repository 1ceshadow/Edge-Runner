## 📖 项目简介

Gamejam中的产物，在不断更新中……

主题："闪“

俯视角跑酷砍杀游戏。角色通过“闪”以及和速度相关的“时缓”技能躲避敌人攻击，并设法接近敌人将其斩杀。

## 🚀 快速开始

### 环境要求
- Unity 版本：`6000.2.9f1`
- 目标平台：Windows

### 项目目录(示例)
Assets/  
├── Animations/                 # 动画和控制器  
├── Audio/                      # 所有音效和音乐  
├── Materials/                  # 材质  
├── Prefabs/                    # 预制体  
├── Scenes/                     # 所有场景  
├── Scripts/                    # 所有C#脚本，按类别分组  
│   ├── Core/                   # 核心游戏系统  
│   ├── Player/                 # 玩家相关  
│   ├── Enemies/                # 敌人相关  
│   ├── Triggers/               # 触发器和事件（例如：WinTrigger.cs）  
│   └── Input/                  # 输入系统  
├── Settings/                   # Unity设置和渲染管线  
├── Sprites/                    # 所有图像、精灵和UI元素  
│   ├── Tiles/                  # Tilemap精灵和切片  
│   ├── UI/                     # UI图像  
│   └── Characters/             # 角色/敌人精灵（例如：来自Picture的玩家精灵，敌人图像）  
└── TextMeshPro/                # Unity包文件夹（自动生成）；
